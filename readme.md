# DuoGastrô 🍽️

> Uma pitada de sal, um toque de açúcar.

Site de receitas artesanais com pratos salgados e doces, desenvolvido com HTML e CSS puro.

---

## 📄 Páginas

| Página                 | Descrição                               |
| ---------------------- | --------------------------------------- |
| `index.html`           | Página inicial com os cards de destaque |
| `páginas/Salgado.html` | Receita de Fricassê de Frango Cremoso   |
| `páginas/Doce.html`    | Receita de Bolo de Morango              |

---

## 🗂️ Estrutura do projeto

```
duogastro/
├── index.html
├── style.css
├── img/
│   ├── Fricasse-de-frango-cremoso.jpg
│   └── Bolo ninho com morango.jpg
│   ├── Bolo de morango.jpg
│   └── fricasse.webp
│   └── Duo.png
└── páginas/
    ├── Salgado.html
    └── Doce.html
```

---

## 🛠️ Tecnologias

- HTML5
- CSS3 (Flexbox, variáveis, pseudo-elementos)
- Google Fonts (Poppins, Montserrat)

---

## ✨ Funcionalidades

- Layout responsivo
- Header fixo com navegação
- Hero com imagem e gradiente
- Passo a passo das receitas com bolinhas numeradas
- Footer centralizado com links de navegação

---

## 🚀 Como rodar localmente

1. Clone o repositório

```bash
git clone https://github.com/Erickhitman/duogastro.git
```

2. Abra o `index.html` no navegador — sem necessidade de servidor ou instalação.
